## Made with

  - React 
  - Redux 
  - Node
  - Express
  - PostgreSQL 
  - Sequelize


## Functions

Using the external [restcountries](https://restcountries.eu/) API we can:

  - Search Countries
  - Filter / Order
  - Create touristic activities
